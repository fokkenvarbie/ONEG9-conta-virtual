<h1>Conta Virtual</h1>
Um programa em Java que simula o funcionamento básico de um sistema bancário. Nele, o usuário pode consultar o saldo, realizar transferências e receber valores em uma conta fictícia, tudo digitando opções numéricas no menu exibido no terminal.<br>
<Br>

- **Funcionalidades:**
  - **Exibição de dados do cliente**: Mostra nome, tipo de conta e saldo inicial ao iniciar o programa.
  - **Menu interativo:** Apresenta opções numéricas para o usuário escolher o que deseja fazer.
  - **Consulta de saldo:** Permite verificar o saldo atual da conta.
  - **Transferência:** Solicita o valor a ser transferido e atualiza o saldo, também garantindo que não seja possível transferir mais do que o disponível.
  - **Recebimento de transferência:** O usuário informa um valor que será somado ao saldo da conta.
  - **Validação de entrada:** Exibe mensagem de erro caso o usuário digite uma opção inválida.
  - **Encerramento seguro:** O programa permanece em execução até o usuário escolher a opção “Sair”.
